# Daily Reflection Tree

This project implements a deterministic reflection system based on structured decision trees.

## Files

* reflection-tree.json → Contains the decision tree
* write-up.md → Design explanation

## How it works

The system guides users through three stages:

1. Locus of Control
2. Contribution
3. Perspective

Each question has predefined options, and each path is deterministic.

## Note

No AI is used in runtime. The system ensures consistent outputs for the same inputs.
